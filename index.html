Typechecking

Types and Sets

  • Each value is it's own type
  • Types can be clustered into other types
  • Hindley-Milner (1969)
  • Curry-Howard-Correspondence (1934, 1958, 1969)

Type union

  • type FooBar = 'foo' | 'bar';
  • type Thing = string | number;
  • type MaybeString = ?string;
  • Also known as sum type

Tuples

  • type Vector2D = [number, number];
  • Also known as product type
  • Bonus level: named tuples: 
    type Vector2D = {
	x: number,
	y: number,
};

Type intersection

  • Type extension: 
    type Vector3D = {
	z: number,
} & Vector2D;
  • Constraint tracking: 
    type Foo = FooBar
& 'foo';
