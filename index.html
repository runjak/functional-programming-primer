Ein paar Worte zu Lisp

  • Erfunden 1958 von John McCarthy
  • LISt Processor
  • S-Expressions: Atome und Listen

Etwas JavaScript

function abs(x) {
  if (x >= 0) {
    return x;
  }

  return -x;
}

Ein AST

Lisp S-Expressions

(defun abs (x)
  (if
    ((>= x 0) x)
    (true -x)
  )
)

Was da los?

  • Eine Sprache aus der Ecke alles ist besteht aus $x.
  • $x sind hier Listen
  • → Komposition
  • → Selbstmodifikation
